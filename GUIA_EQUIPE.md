## ✅ Sistema Está no Ar!

### 🌐 Como Acessar

Acesse pelo navegador:
**https://sistema.apilucenera.site**

---

## 📋 Como Usar (3 Passos Simples)

### 1️⃣ Fazer Upload

1. Acesse **https://sistema.apilucenera.site**
2. Clique em **"Escolher arquivo"**
3. Selecione o arquivo **XML** do orçamento
4. Clique em **"Processar Orçamento"**

⚠️ **IMPORTANTE**: Use APENAS arquivos XML (não PDF)

---

### 2️⃣ Aguardar Processamento

Você verá uma barra de progresso com as etapas:
- 📤 **Uploading** - Enviando arquivo
- ⚙️ **Processing** - Lendo produtos do XML
- 🎨 **Generating** - Criando apresentação PowerPoint
- ✅ **Completed** - Pronto!

⏱️ **Tempo estimado**: 1-2 minutos

---

### 3️⃣ Baixar PowerPoint

Quando concluído, aparecerá o botão:
**📥 Download PowerPoint**

Clique e salve o arquivo gerado!

---

## ✨ O Que o Sistema Faz?

Para cada produto no orçamento XML, o sistema:

1. ✅ Busca informações no Excel Master
2. ✅ Busca imagens técnicas no SharePoint
3. ✅ Cria slide com:
   - Nome do produto
   - Código de referência
   - Código interno
   - Imagens técnicas (quando disponíveis)
   - Informações do fabricante

---

## 🐛 Problemas Comuns

### ❌ "Erro ao processar arquivo"
- **Causa**: Arquivo não é um XML válido
- **Solução**: Verifique se o arquivo tem extensão .xml

### ⏳ "Processamento muito lento"
- **Causa**: Muitos produtos no orçamento (>50)
- **Solução**: Aguarde pacientemente, pode levar até 3-5 minutos

### 🖼️ "PowerPoint sem imagens"
- **Causa**: Produto não tem ficha técnica no SharePoint
- **Solução**: Normal - o slide será criado com placeholders
