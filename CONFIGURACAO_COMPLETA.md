### 1. **Instalar Dependências** (se ainda não instalou)
```bash

pip install -r requirements_sharepoint.txt
```

### 2. **Criar/Preencher Excel Master**
O sistema criará automaticamente em:
```
C:\Users\pedro\OneDrive\Desktop\lucenera\master_produtos.xlsx
```
**IMPORTANTE:** Preencha com dados reais dos seus produtos!

### 3. **Executar Sistema**
```bash
python app.py
```
**Acesse:** http://localhost:5001
